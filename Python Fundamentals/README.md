This folder contains two projects
# A Simple Calculator
The calculator performs arithmetic operations on numbers and can also calculate the factorial of a given number.

## How it works
This script uses python's builtin functions to collect and parse  user input
and perform selected mathematical operationsn on them.
Arithmetic operations that can be performed with the script include:
Addition,Subtraction,Multiplication,Division of two or more inputs and Factorial.

# A Number Guessing Game
This script propmpts the player to guess a number which is then checked against te computer's guess
which is a number generated at random from 1 to 100.

## How it works
The script uses python's built in random module to generate a number at random which the player will then have to guess.
